# VuePress Starter

## points
 <!-- - transpiler changing to babel -->
 - vuepress version is 0.14.2
 - use Vuex
 - use babel-polyfill
 - use svg inline loader
 - use webpack-glsl-loader
 - template engine is pug (pug-plain-loader)
 - simple custom theme

## require
yarn


## start
```
yarn install
yarn dev
```

## build
```
yarn build
```
